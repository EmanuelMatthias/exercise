Projekt: Euler IHK Übung.
Start mit Aufgabe: 1.4 Problem 4: Palindrom-Produkte
Aufgabenstellung:
Ein Palindrom ist eine Zahl, die vorwärts und rückwärts gelesen gleich ist (z. B. 9009 oder 12321).
Finde das größte Palindrom, das als Produkt zweier dreistelliger Zahlen entsteht.
Randbedingungen:
Dreistellige Zahlen sind Zahlen von 100 bis 999.
Das Ergebnis soll das größte Palindrom sein, das sich als Produkt zweier solcher Zahlen bilden lässt.
Beispiel (vereinfachtes Prinzip):
91 × 99 = 9009 → 9009 ist ein Palindrom.
Abgabe:
Gib die gefundene größte Palindrom-Zahl an.
Gib auch die beiden Faktoren an, die dieses Produkt erzeugen.
Zeiteinschätzung (IHK-Prüfung):
Realistisch: 20–30 Minuten
Begründung:
Die Aufgabe erfordert korrektes Schleifenverständnis, bedingte Prüfungen und etwas Optimierung. In einer IHK-Prüfung wäre sie im mittleren Schwierigkeitsbereich einzuordnen.
